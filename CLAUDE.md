### コード変更後のGitHubプッシュ
コードに変更を加えるたびに、以下の手順でGitHubにプッシュすること。

1. 変更ファイルをステージング: `git add <変更ファイル>`
2. コミット: `git commit -m "<変更内容を表すメッセージ>"`
3. プッシュ: `git push origin main`
